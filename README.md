\## Opis projektu

Aplikacja typu REST API zbudowana w oparciu o framework Spring Boot.



\## Uruchomienie i testowanie

1\. Projekt wykorzystuje bazę danych w pamięci H2.

2\. Po uruchomieniu aplikacji (`./mvnw spring-boot:run`), baza inicjalizuje się automatycznie testowymi danymi z pliku `data.sql`.

3\. Dokumentacja Swagger UI dostępna jest pod adresem: 

&#x20;  `http://localhost:8080/swagger-ui/index.html`
